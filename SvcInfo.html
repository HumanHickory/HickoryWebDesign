Website FAQs

Q&A

What is web design?

Web design is the creation of web pages using standardized 'languages'.


What are languages?

Languages, when it comes to web design are programming languages. Basically, they're languages that have been created to communicate with the computer. There are many languages out there that all have different purposes.


What languages do websites use?

Websites use all sorts of languages. The most well-known is HTML5. That's what web pages are actually written in. However, we also use CSS to style the website. Notice that the navigation bar, headers, text fonts and colors are the same on all of my web pages (and almost every website out there)? We use CSS to create conformity.

JavaScript is another popular language for web design and can be written with HTML or using a seperate sheet. JavaScript let's you 'do' certain things, like change margins or create pop-ups while someone is on the page.

These three are the most fundamental languages of web design. Other languages, such as SQL and PHP require servers and databases to work with information. Adding SQL and PHP to websheets not only requires the addition of two more languages, but also requires setting up a seperate server. With that said, SQL and PHP allow for some really cool stuff to be done with a website.


Is it easy to do?

Yes and no. If you are very passionate about it then, with time and practice, it becomes very easy. However, be prepared to spend hours upon hours figuring out how to do one small thing in the beginning.


Where do you build the website?

The website will be built offline in a text editor. Together, we will be able to choose colors, fonts, images then make adjustments until it is perfect. All of this is done, again, offline, so clients will not have access to a half-created website.


What happens after it is ready?

Once the site is ready we will have to choose a host. A host is a site that stores your files and puts them online for you. If your website is just HTML, CSS and JavaScript, there are free options for you. However, if you need a website that stores customer information and/or makes sales, there will be other fees included.

Next, you will need to purchase a domain name if you want something that is not generic. A domain name is the thing you see in the URL. For example, "HickoryWebDesigns.com". "HickoryWebDesigns" is my domain name, that has been purchased. Since I own it, no one else can use it. This prevents traffic confusion; can you imagine going to a URL and getting a different website each time! A generic name is usually created for you for free with a hosting service, but most companies prefer to have their own. Domain names, for the most part, are pretty inexpensive.


What if I don't understand any of this?

That's what we're here for! All of our web design packages include the creation of the website, uploading it to a host and adding a domain name. However, some packages do not include the hosting or domain name prices.


How do I get a logo?

The best way to get a logo is to make one yourself. There are a few programs out there that will allow you to do this. If you are not confident creating one yourself, then finding a graphic artist or using a logo design website is your best bet. Make sure your logo is trademarked!
